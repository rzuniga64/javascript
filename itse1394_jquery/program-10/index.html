1. Create a file named program-10.txt with the following content:
<div>
<p id="p1"> This is p1. </p>
<p id="p2"> This is p2. </p>
</div>

2. Using the program-10.txt file content, create a button with a click event that uses jQuery and AJAX to load only the <p> with id=”p2” into a <div>. Include a callback function that produces an alert with a successful message or error message based on the status.

Let jQuery AJAX change this text.

2. Create a file named program-10.php with the following PHP code:

<?php
$name = $_POST["name"];
$country = $_POST["country"];
echo "Hello ".$name.".";
echo "You were born in ".$country.".";
?>

The program-10.php will be the ‘form processor’ used in the next item. Create a button with a click event that uses jQuery and AJAX to post sample name and country data to the server. Output the two “echo” statements in the program-10.php file to an alert or other suitable mechanism.
